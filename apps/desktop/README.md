gauzy desktop with nstudio electron

how to:

-   install dependencies

```bash
 yarn install
```

-   serve app

```bash
yarn serve:desktop
```

-   build local

```bash
yarn build:desktop:local
```

-   build linux

```bash
yarn build:desktop:linux
```

-   build windows

```bash
yarn build:desktop:windows
```

-   build mac

```bash
yarn build:desktop:mac
```
